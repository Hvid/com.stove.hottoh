Forbind din HottoH træpilleovn til Homey og nyd problemfri kontrol og overvågning. Denne app lader dig tænde eller slukke for din ovn, justere varmeeffekten, indstille måltemperaturer og overvåge vigtige målinger som rum- og røgtemperatur - alt direkte fra Homey. Opret smarte varmeautomatiseringer baseret på dine daglige rutiner, tilstedeværelsesregistrering eller vejrforhold, hvilket gør din opvarmning både mere komfortabel og energieffektiv.
